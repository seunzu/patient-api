| 도구 | 활용 내용 |
|---|---|
| Claude (claude.ai) | 아키텍처 설계 검토, 코드 리뷰, 예외 처리 패턴, 12-Factor App 원칙 적용 |

## AI가 생성한 코드에서 직접 수정한 부분

1. **설계 방향 제시**: 패키지 구조, 레이어 분리, 인터페이스 분리 기준 등 설계 근거를 직접 질문하고 판단
2. **코드 작성**: 설계 방향을 바탕으로 AI가 코드 초안 작성
3. **직접 검토 및 수정**: 생성된 코드를 이해하고 필요한 부분 직접 수정
